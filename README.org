** 문제 해결을 위한 6단계 
   1. 문제를 읽고 이해한다.
   2. 문제를 익숙한 용어로 재정의한다.
   3. 어떻게 해결할지 계획을 세운다.
   4. 계획을 검증한다.
   5. 프로그램을 구현한다.
   6. 어떻게 풀었는지 돌아보고, 개선할 방법이 있는지 찾아본다.
